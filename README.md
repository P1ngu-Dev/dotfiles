# dotfiles

## Instalación en Arch Linux limpio

### 1. Instalación del sistema (desde ISO)

```bash
curl -sL https://raw.githubusercontent.com/P1ngu-Dev/dotfiles/main/install/iso/iso-install.sh | bash
```

### 2. Post-install bootstrap (después de reiniciar)

```bash
curl -sL https://raw.githubusercontent.com/P1ngu-Dev/dotfiles/main/install/arch.sh | bash
```
